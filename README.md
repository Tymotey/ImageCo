# ImageCo
Windows build for ImageCo

## Description
Bulk image Web optimization software. Choose your local folder to optimize and where to optimize it.

## imagesco-settings.json example
```Json
[{
    "id": "folderConvertTest",
    "dirWatch": "D:\\_ImagesCo\\_to_convert",
    "dirTo": "D:\\_ImagesCo\\_converted",
    "processType": "optimize",
    "processMode": "auto",
    "processStatus": 0,
    "watcherFiles": ""
}]
```
